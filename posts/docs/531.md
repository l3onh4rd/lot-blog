# Links für Überschriften in Google Docs

Im heutigen [Video](https://youtu.be/o16_arLQmNE) zeige ich dir die neue Möglichkeit Links für Überschriften zu generieren in Google Docs. 

## Das Wichtigste in Kürze

Diese neue Funktion ermöglicht es dir effizienter in Docs zu arbeiten und schneller die richtigen Stellen deines Dokumentes zu teilen.

## Links für Überschriften

Erstelle dir eine Überschrift mithilfe der Formatvorlagen von Google Docs. Sobald du eine Überschrift erstellt hast, kannst du einen Rechts-Klick auf die machen. In dem sich öffnenden Menü findest du den Punkt "Überschriftenlink kopieren". Klickst du auf diese Option wird ein Link zu dieser Überschrift kopiert. Teilst du diesen Link wird die Person, die den Link öffnet, direkt zu der Überschrift geführt. Dadurch muss man nicht mehr nach bestimmten Stellen im Dokument suchen. Dies funktioniert aber nur solange dein Dokument für die entsprechende Person freigegeben ist. Eine Alternative dazu sind [Lesezeichen](https://youtu.be/V1A4c-whRHg). Mehr dazu erfährst du im verlinkten Video.
